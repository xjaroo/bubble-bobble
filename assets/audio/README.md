# Audio Setup

Put your legally owned audio files in this folder.

The game auto-detects the first playable BGM file from this list (top to bottom):

- `custom-bgm.mp3`
- `custom-bgm.ogg`
- `custom-bgm.wav`
- `custom-bgm.m4a`
- `bgm-custom.mp3`
- `bgm-custom.ogg`
- `bgm-custom.wav`
- `bgm-custom.m4a`

- `bubble-bobble-original.mp3`
- `bubble-bobble-original.ogg`
- `bubble-bobble-original.wav`
- `bubble-bobble-original.m4a`
- `bubble-bobble-original.webm`
- `bubble-bobble-theme.mp3`
- `bubble-bobble-theme.ogg`
- `bubble-bobble-theme.wav`
- `bubble-bobble-theme.m4a`
- `bubble-bobble.mp3`
- `bubble-bobble.ogg`
- `bubble-bobble.wav`
- `bubble-bobble.m4a`
- `bgm.mp3`
- `bgm.ogg`
- `bgm.wav`
- `bgm.m4a`

Background music is looped automatically.

## Bubble SFX (optional)

If present, these sample files are used instead of synth effects:

- Shoot SFX:
  - `sfx-shoot.wav`
  - `sfx-shoot.ogg`
  - `sfx-shoot.mp3`
  - `shoot.wav`
  - `shoot.ogg`
  - `shoot.mp3`
  - `bubble-shoot.wav`
  - `bubble-shoot.ogg`
  - `bubble-shoot.mp3`

- Pop SFX:
  - `sfx-pop.wav`
  - `sfx-pop.ogg`
  - `sfx-pop.mp3`
  - `pop.wav`
  - `pop.ogg`
  - `pop.mp3`
  - `bubble-pop.wav`
  - `bubble-pop.ogg`
  - `bubble-pop.mp3`

If files are missing, the game falls back to built-in synth SFX/BGM.

`M` key mutes both SFX and BGM.
